# Premium QR Code Master 📱

**Premium QR Code Master**는 별도의 설치 없이 브라우저에서 바로 사용할 수 있는 올인원 QR 코드 생성 및 스캔 도구입니다. 최신 웹 트렌드인 **Glassmorphism(글래스모피즘)** 디자인을 적용하여 심미적으로 우수하며, 다크 모드와 라이트 모드를 모두 지원합니다.

> 🚀 **[실행 가능한 데모 사이트 바로가기](https://happyretire.github.io/PremiumQRCodeGenerator/)**

## ✨ 주요 기능 (Key Features)

### 1. QR 코드 생성 (Generator)
- **간편한 생성**: URL이나 텍스트를 입력하고 버튼만 누르면 즉시 QR 코드가 생성됩니다.
- **이미지 다운로드**: 생성된 QR 코드를 고화질 PNG 이미지로 저장할 수 있습니다.
- **실시간 반응**: 빠른 처리 속도로 쾌적한 사용자 경험을 제공합니다.

### 2. QR 코드 스캔 (Scanner)
- **이미지 업로드**: PC에 저장된 QR 코드 이미지를 업로드하여 내용을 확인할 수 있습니다.
- **드래그 앤 드롭**: 파일을 끌어다 놓는 직관적인 방식으로 스캔이 가능합니다.
- **자동 링크 인식**: 스캔 결과가 웹사이트 주소(URL)일 경우, 자동으로 클릭 가능한 링크로 변환됩니다.

### 3. 사용자 경험 (UX/UI)
- **다크/라이트 모드**: 사용자의 환경이나 취향에 맞춰 테마를 자유롭게 전환할 수 있습니다. (설정 자동 저장)
- **프리미엄 디자인**: 부드러운 그라데이션과 블러 효과가 적용된 세련된 인터페이스를 제공합니다.
- **반응형 웹**: 데스크탑뿐만 아니라 태블릿, 모바일 환경에서도 최적화된 화면을 보여줍니다.

## 🛠 사용된 기술 (Tech Stack)

이 프로젝트는 외부 의존성을 최소화하여 `index.html` 단일 파일로 동작하도록 제작되었습니다.

- **Core**: HTML5, CSS3, Vanilla JavaScript
- **Icons**: [Lucide Icons](https://lucide.dev/) (가볍고 예쁜 벡터 아이콘)
- **Libraries**:
  - [QRCode.js](https://davidshimjs.github.io/qrcodejs/) (QR 생성)
  - [jsQR](https://github.com/cozmo/jsQR) (QR 스캔)
- **Fonts**: [Google Fonts (Outfit)](https://fonts.google.com/specimen/Outfit)

## 🚀 실행 방법 (How to Run)

이 프로젝트는 별도의 서버 설정이나 빌드 과정이 필요하지 않습니다.

1. 프로젝트 폴더 내의 `index.html` 파일을 찾습니다.
2. 파일을 더블 클릭하여 크롬(Chrome), 엣지(Edge), 사파리(Safari) 등 웹 브라우저에서 엽니다.
   *(단, 인터넷 연결이 필요합니다. CDN을 통해 라이브러리를 로드합니다.)*

## 📖 사용 가이드

1. **테마 변경**: 우측 상단의 해/달 아이콘을 클릭하여 다크 모드와 라이트 모드를 전환하세요.
2. **모드 전환**: 화면 중앙의 탭 메뉴를 눌러 '생성하기'와 '스캔하기' 모드를 바꿀 수 있습니다.
3. **QR 생성**: 텍스트 입력창에 내용을 적고 엔터키를 치거나 '생성' 버튼을 누르세요.
4. **QR 스캔**: '스캔하기' 탭에서 이미지를 클릭해서 선택하거나 박스 안으로 드래그하세요.
